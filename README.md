### Tech Stack

- **HTML5 / Tailwind CSS** (via CDN) — layout and styling
- **Vanilla JavaScript** — language switching, scroll-activated navigation, floating contact dock
- **Google Fonts** — Space Grotesk & Inter
- **Material Symbols** — icon set
- **GitHub Pages** — hosting

---

### Features

- One-click **English / Chinese** language toggle — all content switches without a page reload
- Responsive layout — sidebar collapses to a top bar on mobile
- Active navigation highlighting driven by IntersectionObserver
- Floating contact button with expand/collapse animation
- Dark-mode-first design using a Material Design 3 colour palette

### 技术栈

- **HTML5 / Tailwind CSS**（CDN 引入）——布局与样式
- **原生 JavaScript**——语言切换、滚动导航激活、悬浮联系栏
- **Google Fonts**——Space Grotesk 与 Inter 字体
- **Material Symbols**——图标
- **GitHub Pages**——托管

---

### 功能亮点

- 一键切换**中文 / 英文**，无需刷新页面，全部内容即时更新
- 响应式布局——在移动端侧边栏自动折叠为顶部导航栏
- 基于 IntersectionObserver 实现滚动时自动高亮当前导航项
- 悬浮联系按钮，支持展开 / 收起动画
- 深色优先设计，采用 Material Design 3 调色板
